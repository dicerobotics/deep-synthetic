# deep-synthetic
- Reduce domain gap between synthetic and real image datasets while preserving features.

인공지능 분야에서 활발하게 연구되고 있는 딥러닝 (deep learning) 기반 영상 분석 및 영상 개선 기법을 이용해서, 유도탄 모의비행시험에 활용되는 가상 적외선 영상 환경에 대한 합성 센서 영상을 보다 사실적으로 생성하는 것을 목표로 다음과 같은 연구를 수행함
(The following study was conducted with the goal of generating more realistic synthetic sensor images of the virtual infrared image environment used in guided missile flight tests using deep learning-based image analysis and image enhancement techniques that are actively being studied in the field of artificial intelligence).

연구목표 (Reserach Objectives):
- 실감 합성 센서 영상 생성기법 연구 (Realistic synthetic sensor image generation technique)
- 실감 합성 센서 영상과 실제 센서 영상 인공지능 기반 비교분석기법 연구 (A research on AI-based comparative analysis technique of realistic synthetic sensor images and real sensor images)
